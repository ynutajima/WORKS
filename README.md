# COOK SNS

【テーマ】<br>
料理をテーマにしたSNS<br>
自炊を応援するサービス<br>

【概要】<br>
自炊を頑張りたくなるようなサイト。<br>
作った料理の写真をSNSに投稿することで、いいねを貰ったり、他者の料理を見ることでモチベーションUP<br>
また、自炊の記録を残すことで、節約度合いなどを自動で計算し、グラフで見える化することでモチベーションUP<br>

【機能】<br>
・投稿機能(文章、写真)<br>
・投稿記録(My記録)<br>
・節約記録グラフ(MY記録)<br>
・投稿記録(全ユーザーの記録)<br>
・ランキング(いいね獲得数)<br>
・お気に入り表示<br>
